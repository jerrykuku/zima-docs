
# Zima Docs Official Website


The website for Zima Docs. Bulit with Hexo.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/jerrykuku/zima-docs.git
$ cd zima-docs
$ yarn install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
